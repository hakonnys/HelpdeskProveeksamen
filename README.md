# Helpdesk System

Dette er et enkelt helpdesk-system der brukere kan registrere seg, logge inn og opprette support-tickets.
Admin-brukere kan se alle tickets og administrere dem, mens vanlige brukere kun kan se sine egne tickets.

Systemet er laget som en full stack-løsning med backend, frontend og lokal nettverkskommunikasjon.



# Funksjonalitet

## Brukere
* Registrere ny bruker
* Logge inn
* Opprette tickets
* Se egne tickets


## Admin
* Se alle tickets
* Lukke tickets
* Opprette admin-bruker med hemmelig kode



# Teknologier

## Frontend
* HTML
* CSS
* JavaScript

## Backend
* Python
* FastAPI
* SQLite

## Sikkerhet
* bcrypt (passord hashing)
* Rollebasert tilgang (admin / user)
